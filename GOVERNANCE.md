# Governance

## Project model

This is a solo-maintainer project hosted under the `clouatre-labs` GitHub organization.

## Roles

- **Project owner**: holds organization-level GitHub access; has final say on all decisions, including releases, roadmap, and breaking changes. The current holder is the owner of the `clouatre-labs` GitHub org.
- **Contributor**: anyone who has had a pull request merged into the repository.

## Decision process

Community input is welcome via issues and pull requests. The project owner makes final decisions on direction, accepting contributions, and releases. See [CONTRIBUTING.md](CONTRIBUTING.md) for how to contribute.

## Access continuity

The project lives under the `clouatre-labs` GitHub organization. Organization ownership can be transferred via GitHub's org transfer mechanism if needed. The Apache 2.0 license guarantees that any fork can continue the project independently, regardless of the state of this repository.

## Bus factor

This is a single-maintainer project. The Apache 2.0 license and the GitHub org transfer mechanism together provide continuity -- any interested party can fork or take over the project.
